Adding a Project

You can add a project using the Add Project wizard. This page explains how to specify the project you want to add and how to indicate that you want to copy or reference this project.

Selecting a Project to Add

To specify the project you want to add:
  • Click before .prj in the Project to be added box, and type the path and file name (keeping the .prj extension).

    Or

  • Click Browse, and find the project you want to import using the file chooser.

Choosing to Copy or Reference the Project

You need to indicate whether you want to make a copy of the project in your current portfolio, or whether you want to create a reference to the location of the project in your current portfolio.

To copy the project:

  1. Click the radio button next to "make a copy of this project and place it in the current portfolio."
  2. Click Next.
    The next wizard window is displayed.

To refer to the project:

  1. Click the radio button next to "include a reference to this project in the current portfolio."
  2. Click Finish.
    Java WorkShop adds the project name to the project manager referring to the original location.
