Adding an Existing Portfolio

You can add a portfolio that already exists into JDE. This portfolio may be one that someone else created or one that you previously created but then removed.

To add an existing portfolio:

  1. Click the Project's button on the JDE main toolbar.
    The JDE Project Manager window is displayed.
  2. In the project manager, choose File -> Add -> Portfolio.
    The Add Portfolio dialog box is displayed.
  3. In the Portfolio file box, type the path name of the portfolio, before the .psf extension.
    Or click the Browse button and use the file chooser to select a directory.
  4. Click the Add button.
JDE stores the portfolio's name and maintains a link to the actual portfolio. The added portfolio is now the current portfolio and the added portfolio name is displayed in the project manager. To view the added portfolio, click the + next to My Portfolios in the project manager.
