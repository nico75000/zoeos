Removing a Project From a Portfolio

When you remove a project from a portfolio, Java WorkShop also removes the project name from the project manager. Java WorkShop does not delete the .prj and other project files from your file system.

To remove a project from a portfolio:

  1. Click the Project's button on the Java WorkShop main toolbar.
  2. You cannot remove the current project. If the project you want to remove is the current project, double-click on another one first.
  3. In the project manager, select the name of the project that you want to remove.
  4. Choose File -> Remove Project From Portfolio.

Because Java WorkShop does not delete the project from your file system, you can later import it by choosing File -> Import -> Project.
