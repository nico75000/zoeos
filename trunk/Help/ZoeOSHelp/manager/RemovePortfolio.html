Removing a Portfolio

To remove a portfolio, JDE removes the portfolio name from the project manager, but does not delete the .psf portfolio file from your file system. After removing a portfolio, you can later add it by choosing File -> Add -> Portfolio.

To remove a portfolio from the project manager:

  1. Click the Project's button on the JDE main toolbar.
  2. You cannot remove the current portfolio. If the portfolio you want to remove is the current portfolio, double-click on another portfolio name first.
  3. Click the name of the portfolio you want to remove.
  4. Choose >File -> Remove Portfolio.
