Creating a Portfolio

To create a portfolio:
  1. Click the Projects button on the Java WorkShop main toolbar.
    The Java WorkShop Project Manager window is displayed.
  2. In the project manager, select the portfolio where you want to create the new portfolio.
  3. In the project manager, choose File -> New -> Portfolio.
    The Create Portfolio dialog box is displayed.
  4. In the Portfolio box, type the new portfolio name before the .psf extension.
  5. Click Create.

Java WorkShop creates the portfolio file in the directory you specified. If you do not specify a path name, Java WorkShop creates the portfolio in your jws directory.

Java WorkShop makes the new portfolio the current portfolio and adds its name to the project manager.

The new portfolio is empty. For information on creating projects, see What Is a Project?.
