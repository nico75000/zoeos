Create Portfolio Dialog Box

Use the Create Portfolio dialog box to create a portfolio in your file system.

Portfolio
The path name of the portfolio file including the extension .psf. If you want other JDE users on the Internet to have access to your portfolio, enter a path name that is accessible by your HTTP server. If you do not specify a path name to the portfolio, JDE creates the portfolio in your .jws directory.

Command buttons
Create--Creates the .psf portfolio file in the directory you specify

Cancel--Closes the dialog box without applying your changes

Help--Displays help on the Create Portfolio dialog box
