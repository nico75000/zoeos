Changing the Current Portfolio

To change the current portfolio:
  1. Click the Project's button on the JDE main toolbar.
    The JDE Project Manager window is displayed.
  2. In the project manager, open a portfolio in one of the following ways:
    • Double-click the name of the portfolio you want to view.
    • Select the name of the portfolio you want to view, and choose File->Open Portfolio.
To display the projects in the portfolio, click on the portfolio icon to the left of the portfolio name.

JDE keeps track of the default project in the .psfaux portfolio file. If you have write permission for the portfolio, the default project is updated every time you change the current project. If you do not have write permission, the default project stored in the file does not change.
