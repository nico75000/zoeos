Add Portfolio Dialog Box

Use the Add Portfolio dialog box to add an existing portfolio into the Java WorkShop. When you add a portfolio, Java WorkShop stores the portfolio's name and maintains a link to the actual portfolio.

Portfolio file
The path name of the portfolio file including the extension .psf. This portfolio can be a portfolio either you or someone else created. Clicking the ellipsis button opens a file chooser in which you can select the portfolio file name.

Command buttons
Add--Adds the portfolio and displays its name in the tool bar

Cancel--Closes the dialog box without applying your changes

Help--Displays help on the Import Portfolio dialog box
