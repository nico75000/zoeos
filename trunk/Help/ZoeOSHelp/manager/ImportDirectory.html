Creating a Name and a Source File Location for an Added Project

After indicating the project you want to copy, create a name and source file location for this project.

Naming the Copied Project

You can change the name of the project you are copying.

To rename the project:

  1. Select the default project name.
  2. Type the new name for the project you are copying in the New project name box.

Naming the Project Storage Directory

The project storage directory box lists the directory where Java WorkShop will keep the source files for the new project.

To designate a storage directory:

  • Type the name of the directory where you want Java WorkShop to store the source files for this project in the storage directory box.
    If the new directory does not exist, Java WorkShop will create it.
  • Click Finish.
    Java WorkShop adds the project name to the project manager, and the source editor window displays the source code.
