Preferences Menu

Proxies Use the Proxies dialog box to change proxy servers (proxies).
Version Control Selects a version control system to use with JDE.
Source Editor Sets preferences for the following source editor behaviors:
  • Indentation/tabs
  • Syntax highlighting and colors
  • Font selection
Keyboard View, add, or delete key assignments (bindings) for functional groups within JDE. See Keyboard Preferences for more information.
Console Input Determines how command-line based program input is interpreted in the Console window. See Console Input Dialog Box for more information.
Builder Determines which Java compiler JDE uses to compile your projects. See Builder Preferences Dialog Box for more information.
Show Labels on Toolbar Adds labels beneath toolbar buttons.
