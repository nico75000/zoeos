Help Menu

Help Contents Opens the table-of-contents for online help.
Task-based Help Displays a menu of help topics for common tasks.
Menu Help Displays a list of help topics for Java WorkShop menus.
Java WorkShop Tutorial Starts the Java WorkShop tutorial, a series of eight interactive tutorials that take you step-by-step through the process of creating, building, running, and debugging an applet project.
Registration Opens the Java WorkShop Registration dialog box. Use this dialog box to electronically register your copy of Java WorkShop.
Java API Documentation (1.1) Displays the 1.1 Java API Documentation in the help viewer. You can also access this documentation through the table-of-contents for online help.
Java Language Specification Displays table of contents of the "Java Language Specification" by James Gosling, Bill Joy, and Guy Steele
About Java WorkShop Opens the About Box. The About Box contains the following information:
  • Serial number
  • Version number
  • Photograph of the Java WorkShop development team
  • Proprietary rights notice
