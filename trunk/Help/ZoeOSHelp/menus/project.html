Project Menu

Show Project Manager Opens the Java WorkShop Project Manager.
Edit Opens the Edit Project window. Use the tabs in this window to edit a project's attributes. See Editing Project Attributes for more information.
Build Builds the current project.
Run Runs the current project.
Profile Runs the current project with the profiling tools.
Stop Program Stops running the current project.
Browse Source Viewer control commands. See Browsing Source for more information.
Copy Applet Tag Copies the applet tag from the current applet project.
Compile File Compiles the .prj file.
New Opens the Create Project wizard.
Add Opens the Add Project wizard.
