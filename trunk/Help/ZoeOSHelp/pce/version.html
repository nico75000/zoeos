Version Control

JDE supports the following version control systems (VCS):
  • PVCS (Microsoft Windows 95/NT)

    See PVCS Notes for important information.

  • RCS (Solaris)
  • SCCS (Solaris)
Choose Preferences -> Version Control (in the main JDE menu bar) to select one of these options. Be sure that the VCS commands are in your execution path.

The file version control commands act on the file that is currently open in the editor window. Choose File -> Version to see the version control commands.

The following VCS commands appear in the Version submenu:
Checkout Checks out the file currently open in the editor from the VCS--makes the file writable.
Check In Checks in the file that is currently open in the editor back into the VCS--makes the file read-only.
Check In New If the file that is currently open in the editor is not under the control of a VCS, this command places it under VCS control--makes the file read-only.
Uncheckout Undoes the effect of the Checkout command. Any modifications you made to the file are deleted and the file is checked back into the VCS--makes the file read-only.

PVCS Notes

If you use PVCS with JDE, be aware of the following:
  • You must define a suffix for your archive files. You can define a suffix using the PVCS Version Manager, or you can edit the appropriate configuration file.

    To define a suffix using the Version Manager:

    1. Select the project to configure in the left pane. For example, "Master Project"
    2. Choose Project -> Configure Project
    3. In the Categories box, select New Archives
    4. In the File Extension grid, add the extension ".java"
    5. In the Archive suffix grid, add the suffix "jvva"
    6. Click OK
    7. Click Save

    To define a suffix by editing the configuration file:

    • Typically, you add an entry to the file C:\PVCS\MASTER.CFG. The following is a suggestion for such an entry in C:\PVCS\MASTER.CFG:

      ARCHIVESUFFIX .java = jvva

  • In order to use PVCS with JDE, you must set the variable VCSCFG to the name of your configuration file.
    • On Windows 95 systems, set VCSCFG in AUTOEXEC.BAT. For example:

      SET VCSCFG=C:\PVCS\MASTER.CFG

    • On Windows NT systems, set VCSCFG in the System Control Panel or in AUTOEXEC.BAT.
