Keyboard Preferences

Use the Keyboard Preferences dialog box (Preferences -> Keyboard) to view, add, or delete key assignments (bindings) for functional groups within JDE.

See Keyboard Actions for a description of the different groups and actions.

To view current key bindings:

  1. Select a group from the Group combo box.
  2. Select an action from the Action list. The current key binding values for that actions are displayed in the Current Key Binding(s) list box.
To add new key bindings:
  1. Select a group from the Group combo box.
  2. Select an action from the Action list. The current key binding values for that actions are displayed in the Current Key Binding(s) list box.
  3. Click in the New Key Binding text box to make it active, then press the keys you want bound to the action.
  4. Click the Add button to add them to the Current Key Binding(s) list.
  5. Click OK or Apply to enter your changes.
To delete current key bindings:
  1. Select a group from the Group combo box.
  2. Select an action from the Action list. The current key binding values for that actions are displayed in the Current Key Binding(s) list box.
  3. Select a binding in the Current Key Binding(s) list.
  4. Click the Delete button to remove it from the Current Key Binding(s) list.
  5. Click OK or Apply to enter your changes.
