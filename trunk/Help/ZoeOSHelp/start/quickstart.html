Tutorial One: Creating a Project

This tutorial is designed for new JDE users. In this tutorial, you will learn what a project is and how to create, build, and run a project in JDE.

Tutorial One is divided into five lessons. To get the most out of the tutorial, you should perform the lessons in the order in which they are presented.

  • Important Concepts: Projects and Portfolios gives a complete overview of how projects and portfolios work. You should understand these concepts before going on in this tutorial.
  • Exercise 1: Creating the Blink Project shows you how to create a project that contains an applet. An applet is a small program that runs in a browser that supports the Java language.
  • Exercise 2: Building the Project shows you how to build the applet project.
  • Exercise 3: Running the Project shows you how to run the applet project and display it in an applet viewer.
  • Exercise 4: Collecting Profile Data shows you how to collect profiling information.
