Tutorial Five: Debugging a Project

This tutorial introduces you to debugging in JDE. You will learn how to examine the threads in a program, set breakpoints, and examine your program's data.

The tutorial consists of seven lessons. You should perform the lessons in the order in which they are presented.

  • Important Concepts: Threads briefly defines threads and thread groups. You should understand these concepts before going on to the exercises.
  • Exercise 1: Starting a Debugging Session teaches you how to begin debugging a project.
  • Exercise 2: Setting a Breakpoint explains how to pause the execution of a program at a specific point in the code.
  • Exercise 3: Examining Threads shows you how to view the thread groups, threads, and stack frames that make up a project.
  • Exercise 4: Stepping Through a Method shows you how to step through code.
  • Exercise 5: Evaluating Expressions shows you how to check the value of your program's data.
  • Exercise 6: Ending a Debugging Session shows you how to kill a debugging session.
