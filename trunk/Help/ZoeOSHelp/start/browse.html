Tutorial Four: Browsing a Project

In this tutorial, you will learn how to browse a project. To get the most out of this tutorial, you should perform the exercises in the order in which they are presented.
  • Exercise 1: Viewing the Class Hierarchy teaches you how to traverse the class hierarchy in your project.
  • Exercise 2: Viewing the Methods in Your Project shows you how to list the methods defined in your project. From this list, you will be able to view the source code for each method.
  • Exercise 3: Finding Strings in Your Code shows you how to search your source code for all occurrences of a specified string constant.
