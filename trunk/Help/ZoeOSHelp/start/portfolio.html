Exercise 1: Creating a Public Portfolio

This exercise shows you how to create a public portfolio; other users on the Internet will be able to access the projects in this portfolio.
  1. Choose Portfolio => Create.
    This action opens the Create Portfolio dialog box.

  2. In the text field, enter a portfolio path name accessible by your HTTP server.
    A portfolio name must end in the extension .psf. For example, on Solaris computing environments, to create the portfolio graphics.psf in the directory /smith/public_html/folios, you would enter /smith/public_html/folios/graphics.psf.

  3. Click the Create button.
    The Java WorkShop makes the portfolio the current portfolio. Other WorkShop users on the Internet can view the portfolio you created by entering the portfolio URL in the browser. In the next lesson, you will learn how to copy a project into the portfolio.
