Tutorial Seven: Creating a GUI

The following lessons show you how to build a calculator using the Visual Java GUI builder. You should complete the lessons in the order in which they are presented.
  • Important Concepts: Project Planning and Design defines the goal of this sample project.
  • Exercise 1: Creating the Project describes how to create a new project called Calculator and save it in your file system.
  • Exercise 2: Starting the Layout shows how to set up the layout window, which helps you position components accurately within the calculator window. You will also learn how to edit a component's basic characteristics.
  • Exercise 3: Adding Components shows how to select a component in the GUI Builder palette and place it in the desired location in your application.
  • Exercise 4: Adding Multiple Components shows you how to add a button to your application and then add the same component in other locations.
  • Exercise 5: Finishing the Layout shows you how to preview the layout.
  • Exercise 6: Adding Code shows how to add code that makes the calculator perform the correct operation when the user clicks on a button.
  • Exercise 7: Building and Running the Project shows how to build the Calculator project and then generate an HTML file that runs the project.
