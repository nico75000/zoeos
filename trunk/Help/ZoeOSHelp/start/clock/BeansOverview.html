Tutorial Eight: Creating the Clock Project

This tutorial is designed for users who are new to creating and using JavaBeans components. In this tutorial you will create and build a simple bean project. Then, you will import the bean project into the GUI builder, where you will add beans to the layout grid and modify the look of the bean.

To get the most out of this tutorial, perform the exercises in the order in which they are presented.

  • Important Concepts: Project Planning describes the design of the Clock project.
  • Exercise 1: Copying the Project Source Files provides information on how to copy source files for the tutorial.
  • Exercise 2: Creating the Clock Project explains how to create a bean project containing multiple beans.
  • Exercise 3: Building the Project explains how to build the Clock project and create the JAR file.
  • Exercise 4: Creating the DeskClock GUI Project shows you how to create a standalone GUI project that will use the beans you produced in the preceding exercise.
  • Exercise 5: Importing the Beans describes how to import the Clock, Marquee, and SoundPlayer beans into the component palette.
  • Exercise 6: Designing DeskClock explains how to add your beans to the DeskClock layout.
  • Exercise 7: Editing the Beans shows you how to change some of their attributes.
  • Exercise 8: Adding Operations to Desk Clock shows you how to edit the Group file to have the Clock bean call the Marquee and Sound Player beans, and test the project.
  • Exercise 9: Modifying Clock describes how you can modify the Clock bean that you created in exercise 2.
  • Exercise 10: Updating DeskClock explains how to use the modified version of the Clock bean in DeskClock.
