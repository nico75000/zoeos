Tutorial Three: Fixing Errors in Source Code

The lessons in this tutorial will teach you how to edit the source code in the Blink project and how to rebuild the project. To get the most out of the tutorial, you should perform the lessons in the order in which they are presented.
  • Exercise 1: Editing Source Code introduces you to some of the basic features of the JDE Source Editor.
  • Exercise 2: Fixing Errors in a Build corrects compiler errors found during the build process.
