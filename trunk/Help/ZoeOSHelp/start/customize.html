Tutorial Six: Customizing a Run Page

This tutorial teaches you how to edit the HTML file that runs a project. To get the most out of the tutorial, perform the lessons in the order in which they are presented.
  • Exercise 1: Editing the HTML File shows you how edit the HTML file that Java WorkShop creates for you when you run a project.
  • Exercise 2: Updating the Run Page shows you how to run the edited HTML file, instead of the default file Blink.tmp.html.
