Tutorial Two: Editing Project Attributes

In this tutorial, you will learn what project attributes are, and how to edit the run and build attributes. This tutorial is divided into three lessons. To get the most out of the tutorial, you should first review the description of project attributes; then go through the two exercises.
  • Important Concepts: Project Attributes explains what project attributes are and how they are used in JDE. You should understand these concepts before going on to Exercise 1.
  • Exercise 1: Editing Run Attributes shows how to specify the way the applet is executed.
  • Exercise 2: Editing Build Attributes shows how to specify building differences.
