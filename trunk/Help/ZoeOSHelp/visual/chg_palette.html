Customizing the Component Palette

Use the Import Components to Palette dialog box to:
  • Rearrange components in a palette tab
  • Remove components in a palette tab
  • Customize components in a palette tab
  • Create a new palette tab
  • Rename a palette tab
  • Remove a palette tab
  • Reset the AWT, Panels, and Additional palette tabs to their original contents

See Importing Components for information on importing JavaBeans with the Import Components To Palette dialog box.

To rearrange components in a palette tab:

  1. Choose GUI -> Customize Palette.
  2. If a palette folder is closed, click on its folder to display a list of its components.
  3. Select a component in the list.
  4. Click the up arrow or down arrow to move the component up or down.

To remove a component from a palette tab:

  1. Choose GUI -> Customize Palette.
  2. If a palette folder is closed, click on its folder to display a list of its components.
  3. Select the component in the list by clicking on its icon or its name.
  4. Click Remove.

To customize a component:

  1. Choose GUI -> Customize Palette.
  2. Select the component by clicking on its icon or its name.
  3. Click Customize.
  4. In the dialog box, edit the three text boxes:
    • Icon (Shadow only)--Path name of the icon file to be assigned to the component (not available for JavaBeans)

      The path to the image must be a fully qualified URL (file:/pathname).

    • Tool Tip Label--Label that appears when the pointer is over the component
    • Footer Label--Text you want to display in the Java WorkShop footer when the pointer is moved over the component in the palette

To create a new palette tab:

  1. Choose GUI -> Customize Palette.
  2. Click New Folder below the Palette Components list.
  3. In the Folder box, type the name you want displayed on the palette's tab.
  4. Click OK.

To rename a palette tab:

  1. Choose GUI -> Customize Palette.
  2. In the Palette Components list, click on the name of the palette folder.
  3. Click Customize below the Palette Components list.
  4. In the dialog box, type the new name you want displayed on the palette tab.
  5. Click OK.

To remove a palette tab:

  1. Choose GUI -> Customize Palette.
  2. In the Palette Components list, click on the name of the palette folder you want to remove.
  3. Click Remove below the Palette Components list.
  4. Click Yes in the Removing palette folder dialog box.

To restore the standard GUI builder palette:

  • Choose GUI -> Reset To Factory Palette.

    The AWT, Panels, and Additional tabs provided by GUI builder are restored to their original state.

Click OK in the Import Components To Palette dialog box to set your edits. If you click Cancel, a window opens asking if you want to save your changes. In this case, your edits are not set unless you click OK in the window.
