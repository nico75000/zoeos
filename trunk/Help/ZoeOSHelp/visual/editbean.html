Customizing a Bean

In the GUI builder, you can customize a bean through the attribute editor or through the bean's customizer (if the bean has a customizer).

To customize the bean with the attribute editor:

  1. Double-click the bean in the layout window to open the attribute editor.
  2. Edit the basic attributes of the bean and click OK to establish the changes. If the bean has a customizer, proceed to the next step to access its more complex attributes.
  3. Click Customizer at the bottom of the attribute editor. This button is active only if the bean has a customizer.
  4. Edit the properties you want to modify as directed.
  5. Click OK in the attributes editor to establish the changes.

To customize an invisible bean with the attribute editor:

  1. Select the bean's name from the Windows list to open the attribute editor for it.
  2. Edit the basic attributes of the bean.
  3. Click OK in the attributes editor to establish the changes.

To customize a bean that comes with a customizer:

  1. Double-click the bean in the layout window to open the attribute editor.
  2. Click Customizer at the bottom of the attribute editor. This button is active only if the bean has a customizer.
  3. Edit the properties you want to modify as directed.
