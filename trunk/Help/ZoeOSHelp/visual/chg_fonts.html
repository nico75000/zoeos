Changing the Font Type

  1. Select a component that contains text.
  2. Open the attribute editor for that component by clicking on its button in the main toolbar.
  3. Use one of the following methods to change the font type:
    • Specify the font name, font style, and font size in the font box and click OK. For example,

      name=SansSerif;style=bold;size=14

    • Click on the button to the right of the font box and set the font type, style, and size from the Font dialog box.
The Font dialog box contains the following fields and buttons:

Name
Displays the name of the currently selected font. The information in this field is updated as you make changes. Use the choice menu to the right of the box to select a font.

Style
Displays the style (plain, bold, italic) of the currently selected font. Use the choice menu to the right of the style box to select a font style.

Point Size
Displays the point size of the currently selected font. Use the choice menu to the right of the point size box to choose a point size.

Sample Text Display Pane
Displays the string "Sample Text" in the currently selected font, style, and point size.

Reset to default font
Resets the font to the default.
