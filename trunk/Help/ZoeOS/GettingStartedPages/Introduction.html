Introduction

Welcome to the ZoeOS Getting Started guide

While there is nothing difficult about using ZoeOS, it is advanced software designed to significantly enhance the usability of your Emulator sampler. It is assumed that you have working knowledge using the Emulator from the front panel and it is not in this brief to reproduce or re-present any information in the Emulator manual as written by E-MU. If you are new to the Emulator world and have never read the manual, we suggest taking a look there first - this guide will use Emulator terminology freely with the assumption that the meaning is understood.

This Getting Started guide will provide a general overview of the applications functionality, taking take the approach of providing pertinent information only.

To aid coherency, here is a summary of the most frequent terms that will be used in the documentation:

Emulator The E4 class digital sampler that ZoeOS controls
Remote A reference to the Emulator
Front panel A reference to the front panel and LCD of the Emulator.
System A reference to ZoeOS
Local A reference to ZoeOS
Cache A reference to the local representation of remote data maintained by ZoeOS
ZCommand A command from the hierarchical functionality scheme used in ZoeOS
Target Target (or targets) of a ZCommand - most commonly specified by a selection on some table.
User preset Any preset in the range 0..999
Flash preset Any preset beyond the last user preset
User sample Any sample in the range 0..999
ROM sample Any read-only sample beyond the last user sample
SMDI The SCSI protocol ZoeOS uses to send and retrieve sample data to and from the Emulator
Midi The wire protocol ZoeOS uses to send musical notes to the Emulator
SysEx (System Exclusive) - The sub-protocol of Midi used by ZoeOS to send and retrieve preset data to and from the Emulator
Interface A Midi interface connected to the computer running ZoeOS
Port An in or out port on a Midi interface or an Emulator
Inport An in port on a Midi interface or Emulator
Outport An out port on a Midi interface or Emulator
Channel A channel on a Midi port - each port has 16 channels.
